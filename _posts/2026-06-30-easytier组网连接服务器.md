## 1. 概述
- **<font color=red>目前是实验功能！</font>**
- 在外网环境下，如需更稳定地访问 Yang Group HFNL 服务器，不限制下载速率和短时间多次建立连接，可先使用 `EasyTier` 加入课题组虚拟局域网，再进行 `SSH` 登录；
- `EasyTier` 连接成功后，本机将获得组内网段地址，可直接访问服务器固定地址和跳板机；
- 出于安全考虑，本文不公开组网配置文件，通过群文件获取。

---

## 2. 下载与安装

- `EasyTier` 官方项目地址：<https://github.com/EasyTier/EasyTier>
- `EasyTier` 下载页：<https://github.com/EasyTier/EasyTier/releases>

下载时请选择与自己操作系统对应的版本。对于大多数用户，推荐优先使用带 `GUI` 的桌面版本。

---

## 3. 获取配置

- 组网配置文件不在网站公开发布，请到群内文件获取；
- 请勿将配置文件转发到公开网页、公开仓库或其他非组内渠道；
- 若群文件中的配置更新，请以最新版本为准。

---

## 4. 导入配置

### GUI 版本

- `GUI` 版本可直接导入群内提供的配置文件；
- 导入后启动对应网络实例即可，无需手动填写密钥等敏感信息；
- 如遇到网卡创建或启动失败，可尝试以管理员权限运行。

### 命令行版本

- 命令行版本同样使用群内提供的配置文件；
- 将配置文件保存到本地后，使用 `EasyTier` 可执行文件加载该配置启动。

Windows 示例：

```powershell
.\easytier-core.exe -c .\hfnl.toml
```

Linux / macOS 示例：

```bash
./easytier-core -c ./hfnl.toml
```

## 5. 连接服务器

如下图所示即为连接成功
![登录流程](/assets/images/easytier.png)

`EasyTier` 启动成功后，可访问以下固定地址：

- 服务器：`10.126.126.1`
- 跳板机 `jump_machine`：`10.126.126.100`

例如，直接访问服务器固定地址，可使用：

```bash
ssh 用户名@10.126.126.1
```

**加入同一 `EasyTier` 网络的设备之间通常可以直接互相访问。例如，个人电脑和笔记本同时接入后，记住这些设备分配的内网IP，如10.126.126.x，可在两台设备之间使用 `远程桌面`、`SSH` 或文件共享等方式进行互联。此类访问仅适用于已授权的个人设备和组内资源，如在 HFNL 实验室笔记本可访问科大台式机。**

---

## 6. 常见问题

1. 若 `GUI` 无法启动网络，请确认已正确导入群内配置，并检查是否存在旧配置冲突；
2. 若程序提示网卡创建失败，请以管理员权限运行，或关闭后重新启动；
3. 若组网成功但无法 `SSH`，请先测试是否能够ping通 `10.126.126.100` 或 `10.126.126.1`；
4. 若群文件中的配置失效或连接异常，请联系管理员获取最新配置。
